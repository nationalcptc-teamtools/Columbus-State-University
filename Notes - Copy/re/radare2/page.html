radare2 BINARY

aaaa  ** Analys All

s sym.main   *** seek to the main function

pdf    ** print the disassembly

ood  ** Open the file again but in debug mode
ood Parameter ** Open the file and pass it a parameter.


db    0x00400649 *** set a breakpoint  then follow it by running ood again.

dc   ** Continue past the breakpoint.

dr  *** Displays registers

dr rip=0x0040064b   ** move the rip to where you want it enabling you to move the registry as needed.

VV        *** Entering visual mode     radare2 Binary, aaa, pdf,  s sym.main, VV
